# Infográfico Gerado com IA

Este infográfico foi produzido a partir de conteúdos analisados no NotebookLM,
com o objetivo de sintetizar visualmente os principais conceitos da ciência da aprendizagem,
como Recuperação Ativa, Repetição Espaçada e o Framework MAPS.

O material foi organizado visualmente para facilitar compreensão rápida e retenção.
